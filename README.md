# ProliferatorMk4

为《戴森球计划》添加增产剂 Mk.IV（四级增产剂），更昂贵，效果更强。

## 内容

- 新物品：增产剂 Mk.IV。
- 新科技：增产剂 Mk.IV。
- 前置科技：亚微观量子纠缠、增产剂 Mk.III。
- 科技消耗：五色矩阵各 1000。
- 配方：2 增产剂 Mk.III + 1 奇异物质 -> 1 增产剂 Mk.IV。
- 增产点数：8。
- 基础喷涂次数：120。

## English

Adds Proliferator Mk.IV to Dyson Sphere Program.

Proliferator Mk.IV follows Proliferator Mk.III and Submicroscopic Quantum Entanglement, using Strange Matter to further compress the active ingredients of proliferators. It is a high-tier proliferator option for mid-late and endgame production lines.

## Dependencies

- BepInEx 5.4.17
- CommonAPI 1.6.5
- LDBTool 3.0.3

## Content

- New item: Proliferator Mk.IV.
- New technology: Proliferator Mk.IV.
- Prerequisite technologies: Submicroscopic Quantum Entanglement and Proliferator Mk.III.
- Technology cost: 1000 of each of the five matrix colors.
- Recipe: 2 Proliferator Mk.III + 1 Strange Matter -> 1 Proliferator Mk.IV.
- Proliferator points: 8.
- Base spray count: 120.
